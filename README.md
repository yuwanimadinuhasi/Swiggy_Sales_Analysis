# 🍔 Swiggy Sales Analysis — Python EDA Project

## Overview
Exploratory data analysis on a large-scale Swiggy food delivery dataset 
covering 197,000+ orders across multiple Indian states and cities. 
The goal was to uncover revenue trends, customer preferences, 
and top-performing restaurants, cities, and food categories.

## Dataset
- 197,430 orders
- 10 features including Order Date, City, State, Dish Name, Price, Rating
- Date range covers multiple quarters of delivery data

## Tools & Libraries
- **Python** — core analysis
- **Pandas & NumPy** — data manipulation and KPI calculation
- **Matplotlib & Seaborn** — static visualizations
- **Plotly Express** — interactive charts
- **Google Colab** — development environment

## Key KPIs
| Metric | Value |
|--------|-------|

## Key Insights
- 📍 **Karnataka** is the highest revenue-generating state, with **Bengaluru** 
  leading among all cities
- 📅 **Saturday** records the highest daily revenue, indicating strong 
  weekend demand
- 🥗 **Veg items dominate** with 70.7% of total orders (139,546) vs 
  29.3% Non-Veg (57,884)
- 🏆 **KFC** is the top revenue-generating restaurant across all locations
- 📈 Quarterly trends reveal consistent growth with seasonal peaks
